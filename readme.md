# HDZ Grasp Project

How to clone this repo:

```shell
git clone --recurse-submodules --shallow-submodules https://github.com/MirTITH/hdz_grasp_proj_v2.git
```

How to update submodules:

```shell
git submodule update --init --recursive --depth 1
```
