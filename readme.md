# HDZ Grasp Project

How to clone this repo:

```shell
git clone --recurse-submodules --shallow-submodules https://github.com/MirTITH/hdz-grasp-proj.git
```

How to update submodules:

```shell
git submodule update --init --recursive --depth 1
```
